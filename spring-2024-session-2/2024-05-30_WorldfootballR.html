WorldFootballR

WorldFootballR is part of the sportsdataverse collection of packages. It’s a database access tool which allows you to pull from several different databases. Each of these databases has its own vignette.

  • For functions to extract data from FBref, see https://jaseziv.github.io/worldfootballR/articles/extract-fbref-data.html

  • For functions to extract data from Transfermarkt, see https://jaseziv.github.io/worldfootballR/articles/extract-transfermarkt-data.html

  • For functions to extract data from Understat, see https://jaseziv.github.io/worldfootballR/articles/extract-understat-data.html

  • For functions to extract data for international matches from FBref, see https://jaseziv.github.io/worldfootballR/articles/fbref-data-internationals.html

  • For functions to load pre-scraped data, see https://jaseziv.github.io/worldfootballR/articles/load-scraped-data.html

FB Ref

(From: https://jaseziv.github.io/worldfootballR/articles/extract-fbref-data.html )

FBref, at https://fbref.com/en/ ,

  • has summaries of shots, events, player performance, expected player performance for each game. https://fbref.com/en/matches/780e0bd7/Sao-Paulo-Talleres-May-29-2024-Copa-Libertadores

  • has player histories at the game and league level https://fbref.com/en/players/ad89aa96/all_comps/Juan-Santos-Stats---All-Competitions

  • has team details too https://fbref.com/en/squads/5f232eb1/2019/c24/Sao-Paulo-Stats-Serie-A

Understat

Understat, at https://understat.com , measures details about shots and uses a complicated neural network model to fit the expected number of goals from each shot.

It has at least two drawbacks: The neural network model is opaque and we can’t mess with it, and it doesn’t really know how to handle penalty shots.

” we trained neural network prediction algorithms with the large dataset (>100,000 shots, over 10 parameters for each). ”

We’re going to create a MIMIC MODEL which is a simpler model (in this case a linear regression) that gets answers that are close to the more complicated model. It’s like making a low-resolution copy of the original model, but with the advantages that come from having a linear regression.
